Shorty

shorty makes interfacing with url shortening services easy, so far the following are supported:

  • bit.ly
  • tr.im - only URL API implemented, not media ones
  • is.gd
  • tinyurl.com
  • cli.gs - only shorten implemented, expand not
  • twurl.nl
  • su.pr

Install

  sudo gem install drcapulet-shorty

Usage

Yeah just read the rdocs. But to get started, make sure the gem is installed. Require: 

  require 'shorty'

or with Rails: 

  config.gem 'drcapulet-shorty', :lib => 'shorty'

Possible Additions

  • ow.ly - currently no API, says it is coming soon

Other Possible Gems

  • post.ly - Not really a shortener
  • ping.fm - Not really a shortener
  • ff.im - Not a shortener either
  • yfrog.com - Not a shortener, maybe expand this gem or a media gem. API docs: code.google.com/p/imageshackapi
  • twitpic.com - Same as yfrog. API docs: twitpic.com/api.do
