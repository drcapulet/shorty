= Shorty

shorty makes interfacing with url shortening services easy, so far the following are supported:
- bit.ly
- tr.im

== Install

  sudo gem install drcapulet-shorty

== Usage

* tr.im
* bit.ly

== Possible Additions

* http://yfrog.com
* http://truwl.nl
* http://twitpic.com
* http://ping.fm
* http://ff.im
* http://su.pr
* http://post.ly - http://post.ly/20Yh
* http://tinyurl.com
* http://is.gd
* http://cli.gs
